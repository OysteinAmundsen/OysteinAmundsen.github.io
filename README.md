# OysteinAmundsen.github.io

A personal blog built with Angular and server-side rendered as a static site. Uses an Nx monorepo for workspace management.

**Live:** [oysteinamundsen.github.io](https://oysteinamundsen.github.io)

## Development

```sh
npx nx serve blog   # Dev server with SSR
npx nx build blog   # Production build
npx nx test blog    # Run tests
npx nx lint blog    # Run linter
```
